# Portfolio

My online portfolio, a project I built in my spare time to show off my projects and experience and practice React and Bootstrap.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
npm install
```

## Usage

```bash
npm start
```
